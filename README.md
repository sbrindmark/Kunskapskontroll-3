# Kodanalys

Ett enkelt C#-konsolprogram för hantering av användare. Programmet låter dig lägga till, visa, ta bort och söka användare i en lista med upp till 10 poster.

## Funktioner

- **Lägg till användare:** Ange ett namn för att lägga till en ny användare (max 10 användare).
- **Visa alla användare:** Visar en lista över alla tillagda användare.
- **Ta bort användare:** Ange ett namn för att ta bort en användare från listan.
- **Sök användare:** Kontrollera om en viss användare finns i listan.
- **Avsluta:** Stänger programmet.

## Användning

1. Starta programmet.
2. Välj ett alternativ i menyn genom att ange en siffra (1-5).
3. Följ instruktionerna i konsolen.

## Krav

- **C# version:** 7.3
- **.NET Framework:** 4.7.2

## Kompilering och körning

1. Öppna projektet i Visual Studio 2022.
2. Bygg lösningen (__Build Solution__).
3. Starta programmet (__Start Debugging__ eller tryck F5).

## Filstruktur

- `Program.cs` – Huvudprogramfilen med all logik för användarhantering.
